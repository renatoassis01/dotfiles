### Setup initial

```sh
./setup.sh
```

### Uninstall dotfiles

```
./uninstall.sh
```

### Config git user

```sh
git config --global --add --bool push.autoSetupRemote true
git config -f ~/.gitlocal user.email "email@yoursite.com"
```

```sh
git config -f ~/.gitlocal user.name "Name Lastname"
```
